# Pepper Pi

How to build a floating transparent display for a Raspberry Pi 5.

## Build Video

https://youtu.be/vGyMrAtvg8A

## OS commands
To flip the display for reflection:
```
wlr-randr --output DSI-2 --transform flipped-180
```
